Pseudocoding 1

How to use a fire extinguisher:
Grab… Grab a fire extinguisher.
PULL... Pull the pin. This will also break the tamper seal.
AIM... Aim low, pointing the extinguisher nozzle (or its horn or hose) at the base of the fire.
NOTE: Do not touch the plastic discharge horn on CO2 extinguishers, it gets very cold and may damage skin.
SQUEEZE... Squeeze the handle to release the extinguishing agent.
SWEEP... Sweep from side to side at the base of the fire until it appears to be out. Watch the area. If the fire re-ignites, repeat steps 2 - 4.

Pseudo-code for interactive fire extinguisher model
// 1.Create large display block for simulation
// 2.Enter “click to start” so it can start
// 3.First show fire and make fire extinguisher clickable
// 4.Zoom on extinguisher, and step by step on what to do by clicking the directions
// 5.Make the user wave mouse back and forth to put out virtual fire.
// 6.End by putting “congratulations”
// 7.Test for errors
// 8.Publish

Pseudocoding 2

Creating a interactive rock, paper, scissors game
Enter “click to start” so it can start
Show how much time player has
Click an option rock, paper, scissors
Wait for timer to hit 0
AI will have answer
AI shows answer.
Winner is decided.

Pseudo-code for creating a rock, paper, scissors game:
// 1.Create a start button, if user pushes start then game will begin.
// 2.Set a adjusted time to allow users input.
// 3.Provide three options for user to pick.
// 4.Validate input.
// 5.Once timer hits 0, play AI's answer.
// 6.AI's answer is generated randonmly based on the three available options.
// 7.Based on the users and AI's answer, the game will decide the winner.
